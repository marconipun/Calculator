# Java Calculator - Command Line Application

## Overview
This is a **simple, interactive command-line calculator** built in Java that performs basic arithmetic operations. Users can **add, subtract, multiply, and divide** numbers with an intuitive interface that continues running until they choose to exit.

The project demonstrates fundamental Java programming concepts including methods, user input handling, loops, and conditional statements.

## Features in the app

### 1. Basic Arithmetic Operations
- ➕ **Addition** - Add two numbers together
- ➖ **Subtraction** - Subtract one number from another
- ✖️ **Multiplication** - Multiply two numbers
- ➗ **Division** - Divide numbers with zero division protection

### 2. Interactive Console Interface
- Continuous calculation mode - perform multiple calculations without restarting
- Clear prompts guiding users through each step
- Result display after each calculation
- Option to continue or exit the program

### 3. Error Handling
- **Division by zero protection** - Prevents crashes with friendly error message
- **Invalid operation handling** - Gracefully handles incorrect inputs
- Input validation for numbers and operations

### 4. User Experience
- Simple "y/n" prompt to continue calculations
- Clear display of entered numbers and results
- Friendly closing message when exiting

## Tech Stack that has been used
- **Java**: Core programming language used for all logic and functionality
- **Java Scanner Class**: Used for handling user input from the console
- **Eclipse IDE**: Development environment used to build and test the application
- **Java SE 8**: The Java version used for development

## Instructions about how to run the Program

### Using Eclipse:
1. **Clone the repository** to get the code :[Github]( https://github.com/marconipun/Calculator.git)

2. **Import into Eclipse**
- Open Eclipse IDE
- File → Import → General → Existing Projects into Workspace
- Browse to the project directory and select it
- Click "Finish"

3. **Run the application**
- Navigate to `calculator.java` in the Package Explorer
- Right-click → Run As → Java Application
